Aufgabenstellung

In dieser Aufgabe üben wir "External Style Sheets". Die Anweisungen zum Darstellen der Seite sind in einer einzigen Datei gespeichert, was es wirklich einfach macht, das Styling über eine ganze Website oder mehrere Elemente hinweg zu bearbeiten. Hier ist eine einfache HTML-Seite mit External Style Sheet:

Bitte verwende den folgenden Code und füge die fehlenden Werte zwischen geschweifte Klammern {} hinzu:

  • color: darkgrey
  • color: green
  • background: lightgray
  • font-size: 40px
  • font-size: 35px

Neben index.html erstelle einen Ordner assets dann einen Unterordner css mit einer Datei styles.css darin. Füge diesen Code in style.css ein.
